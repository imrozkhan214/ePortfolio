My Projects

Ustep File Transfer

Technologies: Tkinter, Flask, Selenium, Python

This project is designed to automate the process of downloading course materials from the University of Science and Technology of Southern Philippines (USTP) USTEP portal.

Using a simple drag-and-drop UI built with Tkinter, users can easily upload files and select courses from which they wish to download resources. The project uses Flask for the backend and Selenium for automating web interactions, ensuring that the files are retrieved efficiently and that the file names in the UI match exactly those listed on the USTEP resource page. The solution provides a user-friendly interface that simplifies the downloading process, saving time for students.

FNA String Generator

Technologies: Automata Theory, Python

The FNA String Generator is a project based on automata theory, designed to generate strings according to predefined rules or patterns.

Using a finite automaton (FNA), the system processes input conditions and outputs strings that conform to the specifications of the automaton. The project focuses on the creation of an automaton that can generate strings in a structured, rule-based manner, making it a useful tool for understanding automata theory and its application in string generation. The solution can be applied to various fields such as pattern recognition and language processing.
